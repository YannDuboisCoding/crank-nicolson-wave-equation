# Finite Difference Method — 1D Wave Equation (Crank-Nicolson)

## Description
Numerical study of the 1D wave equation using the Crank-Nicolson finite difference scheme.
Experimental analysis of the L2 convergence order of the scheme.

## Methods
- Finite difference method (Crank-Nicolson scheme)
- L2 error analysis and convergence order estimation

## Technologies
Python · NumPy · Matplotlib
